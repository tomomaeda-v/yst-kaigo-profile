# yst-kaigo-profile

YS Talent 介護候補者プロファイル管理（特定技能介護 人物評価システム）

[yst-fti-profile](https://github.com/tomomaeda-v/yst-fti-profile)（ドライバー版）を基に、特定技能・介護の外国人候補者向けに全面改訂した性格診断アプリです。

## 主な変更点（ドライバー版との差分）

- 特性軸を12→14に拡張：**共感・思いやり（Empati & Kepedulian）**、**介助受容性（Penerimaan Tugas Perawatan）** を追加
- シチュエーション設問を介護場面に全面差し替え（認知症対応・入浴拒否・排泄介助・夜勤・転倒発見・看取り など全8問）
- 高齢者との会話・介護経験に関する設問を追加（全48問）
- リスクフラグに「共感性が低い」「身体介助への抵抗感が強い」等を追加
- 顧客向けアドバイス自動生成を介護文脈に調整
- カラーテーマをティール（緑系）に変更し、ドライバー版と視覚的に区別
- localStorageキーを分離（ドライバー版とデータが混ざりません）

## 使い方

- 管理画面: `index.html` をそのまま開く（GitHub Pages URL）
- 候補者用診断: URL末尾に `?mode=quiz` を付与（インドネシア語＋日本語）

## 初期設定

1. `index.html` 内の `CONFIG.backendUrl` は空です。チーム同期を使う場合は、**介護版専用の** Google Apps Script Web App（ドライバー版と同じスクリプトを新規シートで再デプロイ）のURLを設定してください。未設定でもローカル保存で動作します。
2. 既存メンバーはサンプルデータです。「👥 既存メンバー」タブから実データに置き換えてください。
